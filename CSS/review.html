CSS syntax ประกอบด้วยอะไรบ้าง
    1.Selector คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล
    2.Property คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง
    3.Value คือ ค่าที่เราต้องการให้แสดง
selector {property: value}
